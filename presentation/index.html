What is Pathman?

Pathman is an educational pattern recogntion game which is designed to help fourth graders recognize patterns more quickly and easily. It uses gamification to reinforce concepts learned in school and cultivate problem solving skills.

CORE Curriculum

We based our gameplay off of the 4th grade CORE curriculum surrounding pattern recognition.

GamePlay

  • Player is given a maze with a starting square and an ending square, and has to navigate to the end
  • Player can only move to spaces that follow the current pattern
  • Time is lost if player makes a mistake, and a life is lost if the timer runs out
